# 说明  
 > 首次上线需要执行的命令  
 
 ```bash
 # 1.1 安装node
     安装nodejs
     node-v/npm -v
     
     安装cnpm（安装或者不安装都可以，安装之后用cnpm命令）
     npm install -g cnpm --registry=https://registry.npm.taobao.org
     
 1.2 安装webpack
     npm install webpack -g
     查看版本号：webpack -v
     
     webpack脚手架
     npm install webpack-cli -g
     
 
 1.3 安装vue-cli
   npm install --global vue-cli 
   
   

   
 ```
***

> 2.项目创建
```bash
     2.1 创建一个基于webpack模板的项目
       vue init webpack 项目名称
    
    2.2 切换路径，安装依赖（install dependencies）
        cd 项目名称 
        npm install --registry=https://registry.npm.taobao.org
    
    2.3 部署运行（serve with hot reload at localhost:8080）
        npm run dev
    
    2.4 生产构建 build for production with minification
        npm run build
    
    2.5 build for production and view the bundle analyzer report
       npm run build --report

```     
***     

> 3.服务搭建
```bash
  3.1 ui框架引入
  # element-ui 安装
  $ npm i element-ui -S

  3.2 axios（类似ajax）引入
  # axios安装
  $ npm install axios
  
  3.3 安装后台框架
     npm install express -save
     npm install body-parser
     
  3.4 安装mysql驱动(开发使用MySQL数据库)
     npm install mysql
  
  3.5 less预处理安装
  npm install less less-loader --sava-dev
  
  3.6 全局安装supervisor   
    npm install -g supervisor
   
  3.7 js-cookie安装
    npm install --save js-cookie
  3.8 mockjs 安装
  npm install mockjs --save-dev
  
  3.9 安装vuex和兼容vuex的promise
  npm install es6-promise --save
  
```
> 4 使用的技术
```text
  4.1 前端技术
     vue(javasript框架)

     axios (类似于ajax)
     
     
  4.2 服务器

     mock(模拟测试数据)
      

  4.3 数据库（开发的时候可以使用，现网环境中不需要直接调用数据库）
    mysql
    Mybatis
    
  
  
  
```
